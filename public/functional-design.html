App Overzicht

Wat is PointsFam?

PointsFam is een web-gebaseerde familie management applicatie waarbij kinderen punten kunnen verdienen door het uitvoeren van huishoudelijke taken en het vertonen van goed gedrag.

Hoofddoelstellingen:
  • Motivatie: Kinderen motiveren om taken uit te voeren
  • Structuur: Duidelijke structuur in huishoudelijke taken
  • Beloning: Punten systeem met inwisselbare beloningen
  • Communicatie: Verbeteren van familie communicatie
  • Verantwoordelijkheid: Kinderen leren verantwoordelijkheid nemen
Doelgroep
  • Families met kinderen
  • Kinderen 6-16 jaar
  • Ouders die structuur willen
  • Huishoudens met taken

Technische Architectuur

Backend Stack

  • Runtime: Node.js 18+
  • Framework: Express.js
  • Database: MySQL (Aiven Cloud)
  • Authentication: Express-session
  • File Upload: Multer
  • Security: Helmet, CORS

Frontend Stack

  • Markup: HTML5
  • Styling: CSS3 + Bootstrap 5
  • Scripting: Vanilla JavaScript
  • Icons: Font Awesome 6
  • Responsive: Mobile-first design
  • PWA: Service Worker ready

Database Schema

Gebruikers Tabellen:
users (id, username, password, firstName, lastName, role, familyId, points, profile_image)
families (id, name, created_at)
family_members (family_id, user_id, role, joined_at)
Taken Tabellen:
tasks (id, name, description, points, category, family_id)
task_assignments (id, task_id, user_id, assigned_by, status, points_awarded)
task_completions (id, assignment_id, completed_at, approved_by)
Beloningen Tabellen:
rewards (id, name, description, points_required, category, family_id)
reward_redemptions (id, reward_id, user_id, points_spent, redeemed_at)
Communicatie Tabellen:
conversations (id, title, type, participants, family_id)
messages (id, conversation_id, sender_id, content, sent_at)
direct_messages (id, sender_id, receiver_id, content, sent_at)
